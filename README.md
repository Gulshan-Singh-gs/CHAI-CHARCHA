# ChaiTrade · Skill swaps over chai

Welcome to **ChaiTrade**, a platform for community skill swapping over a cup of chai!

This project is a static website built with HTML, CSS, and JavaScript (bundled via Vite).

## Deployment

The site is configured to automatically deploy to GitHub Pages via GitHub Actions (see `deploy.yml`).

## Running Locally

Because this is a static site, you can view it by simply serving the files locally.

If you have Python installed, you can use the built-in HTTP server:

```bash
python3 -m http.server 8000
```

Then, open [http://localhost:8000](http://localhost:8000) in your web browser.
